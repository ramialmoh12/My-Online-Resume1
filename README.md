# Rami Almohamad - Modern Online CV

A modern animated online CV built with HTML, CSS, and JavaScript.

## How to upload to GitHub Pages

1. Create a new GitHub repository, for example: `rami-online-cv`.
2. Upload these files to the repository:
   - `index.html`
   - `style.css`
   - `script.js`
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. Wait 1–3 minutes. Your website will become live at:
   `https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Edit information

You can update the content inside `index.html`.
